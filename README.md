## Usage

### Step 1

Find a video on Youtube (preferably a music)

### Step 2

In the URL, replace `www.youtube.com` by `waaw.space`

For example

`https://www.youtube.com/watch?v=Y4HWvsGs0rY`

becomes

`https://waaw.space/watch?v=Y4HWvsGs0rY`
